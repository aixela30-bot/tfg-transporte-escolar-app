# TFG - Aplicación de transporte escolar

Este repositorio contiene el desarrollo del Trabajo Fin de Grado de Ingeniería Informática.

## Descripción
Aplicación móvil para mejorar la seguridad en el transporte escolar mediante:
- Control de menores
- Geolocalización en tiempo real
- Notificaciones a familias
